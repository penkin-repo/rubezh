Отличный темп! Логичная структура URL — это фундамент для хорошего сканирования сайта поисковыми ботами и удобства пользователей.

Поскольку в задании есть четкое требование использовать ЧПУ на русском транслите, мы адаптируем структуру так, чтобы она содержала ключевые слова и сохраняла иерархию (вложенность).

Шаг 2.3: SEO-структура URL (ЧПУ)
Вот полная карта URL для сайта ООО "Рубеж":

1. Главная страница

/ — корень сайта.

2. О компании и карьера (Логическая связка)

/o-kompanii/ — основная страница о компании.

/o-kompanii/vakansii/ — вложенный раздел с вакансиями.

Почему так: Вакансии логично относятся к информации о самой организации. Это показывает поисковикам структуру и связывает кадровый потенциал с брендом работодателя.

3. Проекты (Портфолио)

/proekty/ — общая галерея (содержит ключ "проекты").

Динамические категории (для фильтрации и низкочастотных запросов):

/proekty/dorozhnoe-stroitelstvo/

/proekty/asfaltirovanie/

/proekty/inzhenernye-seti/

Пример конкретного объекта:

/proekty/asfaltirovanie/remont-dvora-arkhangelsk/ (В URL сразу зашита геопривязка и вид работ).

4. Аренда спецтехники (Каталог)

/arenda-spectehniki/ — главная страница каталога (содержит высокочастотный ключ).

Категории техники (Вложенность уровня 2):

/arenda-spectehniki/samosvaly/

/arenda-spectehniki/ekskavatory/

/arenda-spectehniki/buldozery/

/arenda-spectehniki/katki/

/arenda-spectehniki/spectehnika/

Карточки конкретной техники (Вложенность уровня 3 — динамический роут):

/arenda-spectehniki/samosvaly/howo-t5g-30t/

/arenda-spectehniki/ekskavatory/hyundai-hx220s/
